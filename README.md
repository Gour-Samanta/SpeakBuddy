# Features list:
- User authentication with JWT
- Find speaking partners
- Real-time chat
- Video calling
- Responsive design

# Tech Stack:
Frontend: React.js  &nbsp;&nbsp;
Backend: Node.js, Express.js  &nbsp;&nbsp;
Database: MongoDB  &nbsp;&nbsp;
Auth: JWT  &nbsp;&nbsp;
